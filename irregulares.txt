// Legenda:
// palavra pessoa sing/plural  [p|
// p - presente do indicativo
// pi- pretérito imperfeito do indicativo
// pp - pretérito perfeito do indicativo
// pmp - pretérito mais que perfeito
// f - futuro
// pc - presente do conjuntivo
// pic - pretérito imperfeito do conjuntivo
// fc - futuro do conjuntivo
// c -condicional
// i - imperativo
// g - gerundio
// ppa - particípio passado
//
// nesta versão são retirados os que estão no U.AFF, e nos X é posto X:letra
// tem de se por pessoa 0 no g (gerundio) e no ppa (particípio passado)

// que nos gerados antes do X há sempre uma letra, que é a que vai ser alterada
// deve-se indicar qual é essa letra (é procurada a partir do fim (sem contar
// com o ar, er, ir, já que sabemos que pertence ao radical.
